Frequently asked questions

What is this?
It's an interactive character sheet for pen and paper role playing games.
It aims to give provide a decent looking sheet on the screen and on paper.
You can edit most fields by clicking on them. Some fields implement the rules of the most famous role playing game. Just give it a try.
Something isn't working right. Can you fix it?
I'll try. Please report any bugs using the feedback page.
If you're having trouble with a specific device or browser, there probably won't be a fix in the near future. Time is sparse.
What kind of devices can I use?
Any reasonably recent device of a certain size. Tablets and desktop computers just have more pixels than most phones. Browsers that should work best are Firefox and Chrome.
When in doubt, give it a try.
I don't like how this looks. Can change some things around?
Probably not, sorry. It's a time issue.
But you or someone you know could check out the code for this app on github and change it to your heart's delight.
Some features are missing.
That's not a question.
Are you gonna add missing features?.
Possibly. If the required time can be found. Some features might never make it into the app though.
Things that probably will not make it anytime soon include:
  • multi-classing
  • player races and player classes that are not part of the PHB
  • multi
Further up the priority list are things like:
  • more layouts and designs
  • making parts of the character sheet optional or adjustable
What does "Save Character" actually do? What about "Clear local data"?
Clicking the "Save Character" button will save the character
Cookies, huh?
This app uses no cookies, no adds, and no trackers of any kind.
When you use the save character feature, a small amount of information will be saved on your device.
This enables the app to restore your changes when you visit again.
While this data can theoretically be used to check whether you have been on this page before, it is not being sent anywhere at any time.
You can delete this piece of information with the "clear local data" button at the bottom of the character sheet. Some devices will clear this data automatically after a certain time. If you want to ensure your changes will not be lost, please use the "export character" feature.
