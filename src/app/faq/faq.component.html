Frequently asked questions

Something isn't working right. Can you fix it?
I'll try. Please report any bugs using the feedback page.
If you're having trouble with a specific device or browser, there probably won't be a fix in the near future. Time is sparse.
What kind of devices can I use?
Any reasonably recent device of a certain size. Tablets and desktop computers just have more pixels than most phones. Browser that should work best are Firefox and Chrome.
When in doubt, give it a try.
I don't like how this looks. Can change some things around?
Probably not, sorry. It's a time issue.
But you or someone you know could check out the code for this app on github and change it to your heart's delight.
Some features are missing.
That's not a question.
Are you gonna add missing features?.
Possibly. If the required time can be found. Some features might never make it into the app though.
Things that probably will not make it anytime soon include:
  • multi-classing
  • player races and player classes that are not part of the PHB
  • multi
Further up the priority list are things like:
  • more layouts and designs
  • making parts of the character sheet optional or adjustable
Are these questions really asked frequently?
Hell nah. But I do hope this list helps.
