# CharSheet
Interactive Character Sheet for Fantasy Role Playing Games.

v.0.0.2

## Dependencies
* Angular
* rxjs
* ngneat/dialog
* FileSaver.js https://github.com/eligrey/FileSaver.js

## How to
1 Grab it
```
git chekout
```

2 Get the dependencies and start it
```
npm install
npm run start
```

3 Visit http://localhost:4200
